# Workout Wednesday 2026 - Week 27 | Sentence Template Tooltips in Power BI

## 📌 Overview

This repository contains my solution for **Workout Wednesday 2026 – Week 27**, a Power BI challenge focused on creating **dynamic sentence-based tooltips** that enhance dashboard storytelling and user experience.

Instead of displaying only numerical values, the tooltip dynamically generates a meaningful sentence using DAX, making it easier for users to interpret the insights.

---

## 🎯 Challenge Objective

Build an interactive tooltip that displays:

* Current Enrollment
* Previous Year's Enrollment
* Year-over-Year (YoY) Percentage Change
* College Name
* Selected Year
* Trend Indicator (▲ / ▼)

---

## 🛠️ Tools & Technologies
